About This Project

The Movie Search App allows users to search for movies and TV shows by fetching data from the TVMaze API. With a simple and intuitive interface, users can enter a movie or show title, and the app will display a list of matching results, including movie posters and names.

This app is built using modern web technologies including HTML, CSS, and JavaScript. Axios is used to make asynchronous requests to fetch data from the API, ensuring a smooth and responsive user experience.

Additional features include animated transitions for displaying movie results and an easy-to-use interface, making this app a great tool for finding information about your favorite movies or discovering new ones.
